# common-security-jwt

## 项目概述

基于 JWT 的认证实现。

## 面向开发者

### 核心功能

- JWT Token 生成与验证
- Token 刷新机制

### 依赖引入

```xml
<dependency>
    <groupId>com.dev.lib</groupId>
    <artifactId>common-security-jwt</artifactId>
</dependency>
```
