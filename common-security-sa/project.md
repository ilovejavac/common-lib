# common-security-sa

## 项目概述

基于 Sa-Token 的认证授权实现。

## 面向开发者

### 核心功能

- Sa-Token 集成
- 权限认证
- 会话管理

### 依赖引入

```xml
<dependency>
    <groupId>com.dev.lib</groupId>
    <artifactId>common-security-sa</artifactId>
</dependency>
```
