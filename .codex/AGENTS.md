# AGENTS.md — PM 技能触发器

> 本文件的**唯一职责**：确保 PM 技能（SKILL.md）被触发。
> 所有规范、流程、约束由 SKILL.md 全权管理。
> 本文件仅在 LLM 未自动加载 SKILL.md 时作为备用。

## 触发规则（必须触发）

**以下任何条件满足时，必须加载并遵循 PM 技能：**

1. 工作目录存在 `.project/` 目录
2. 用户提到项目管理意图（初始化/提议/规划/执行/验证/调试/状态/恢复/继续/做个改动/加个功能）
3. 用户使用 `pm:` 前缀命令
4. 会话开始时（启动协议必须执行 — 含自动初始化）
5. 检测到源码文件但无 `.project/`（自动扫描并构建）

**触发后的行为完全由 SKILL.md 定义。**

## 备用最小规则（仅当 SKILL.md 未加载时）

**语言：** 始终中文回复，代码标识符英文。

**启动：**
1. `.project/` 存在 → 读 STATE.md + PROJECT.md → 报告状态
2. `.project/` 不存在 + 有代码 → 自动扫描构建 `.project/`（不问用户）
3. `.project/` 不存在 + 空目录 → 创建最小 `.project/` → 等待输入

**缓存：** 文件摘要写 `.project/.cache/index.md`，关联写 relations.md，决策写 decisions.md。

**状态：** 每次操作后更新 `.project/STATE.md` 活动日志。

**Git：** `<type>(<scope>): <中文描述>` — 每任务原子提交。
