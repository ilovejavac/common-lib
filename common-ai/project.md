# common-ai

## 项目概述

AI 功能集成，基于 LangChain4j。

## 面向开发者

### 核心功能

- LLM 集成
- 向量存储
- RAG 支持

### 依赖引入

```xml
<dependency>
    <groupId>com.dev.lib</groupId>
    <artifactId>common-ai</artifactId>
</dependency>
```
