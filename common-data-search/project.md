# common-data-search

## 项目概述

OpenSearch/Elasticsearch 搜索集成。

## 面向开发者

### 核心功能

- 全文搜索
- 索引管理

### 依赖引入

```xml
<dependency>
    <groupId>com.dev.lib</groupId>
    <artifactId>common-data-search</artifactId>
</dependency>
```

## 面向 LLM

### 核心组件

基于 Spring Data OpenSearch 2.0.5
